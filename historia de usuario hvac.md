A. Historias de Usuario para Técnicos y Profesionales HVAC/R

| COMO (Usuario) | QUIERO (Acción) | PARA (Beneficio) |
| :---- | :---- | :---- |
| **COMO** Técnico en Mantenimiento Preventivo (PM Tech) \[3, 18\], | **QUIERO** consultar la información sobre **Herramientas y Procedimientos de Calidad** \[4, 19\], | **PARA** asegurar que utilizo la **Bomba de Vacío** (idealmente de dos etapas \[4\]) y que realizo un **vacío profundo** (entre **50 y 200 micras**) \[4\] antes de cargar el gas. |
| **COMO** Técnico especializado \[20\], | **QUIERO** verificar si hay contaminación ácida en el sistema \[3\], | **PARA** realizar la **Comprobación de ácidos** \[3\] con **QwikCheck®** \[3, 20\] y confirmar que no hay ácidos si el indicador no cambia a **naranja o rojo** \[3, 20\]. |
| **COMO** Especialista en Reparación \[3\], | **QUIERO** verificar la integridad eléctrica del compresor \[4, 20\], | **PARA** chequear el **voltaje** de conexión \[4\] y diagnosticar que las **bobinas no estén abiertas** \[20\] utilizando el **Multímetro Digital** \[4\]. |
| **COMO** Técnico que trabaja con alta presión \[4\], | **QUIERO** informarme sobre la técnica segura para buscar fugas \[4\], | **PARA** emplear la **prueba de presión estática con nitrógeno** \[4\] y recordar que **nunca debo utilizar oxígeno o aire** para presurizar el sistema \[4\]. |
| **COMO** Instalador en campo \[21\], | **QUIERO** acceder al sitio web **desde dispositivos móviles** \[22\], | **PARA** consultar los procedimientos técnicos detallados y asegurar el cumplimiento de las normativas mientras **trabajo en el campo** \[21\]. |
| **COMO** Técnico que realiza revisiones eléctricas \[3\], | **QUIERO** conocer la solución para terminales dañados o corroídos en cables conductores \[3\], | **PARA** utilizar **QwikLug®** y realizar una **Revisión Eléctrica** adecuada \[3\]. |
| **COMO** Profesional preocupado por la normativa \[7, 17\], | **QUIERO** leer el **Blog Técnico** o la sección de **Sustentabilidad y Normativa** \[2, 9\], | **PARA** mantenerme actualizado sobre el **Reglamento F-Gas (UE) 2024/573** \[7, 17\] y entender la **eliminación total de HFC para 2050** \[7, 17\]. |
